FAQs

Frequently Asked Questions

How do I set up my Boltcard?

First head to the git link at the bottom of this page and set up your boltcard service as described in the boltcard repository

Order some blank NFC cards online and then download and install the NFC programming app to set up your card

How does bolt card work?

Each NFC card is programmed with your boltcard service's lnurlw:// address and some secure keys that generate a new unique hash each time the card is scanned. This means that the card outputs this LNURL and unique numbers to the merchant.

The merchant can then use these values to make a one time call to your bolt card service, after that the card must be tapped again to get fresh values.

What does a merchant need to do to accept Boltcards?

The merchant just needs to use an android device that supports one of the merchant apps or systems mentioned in the about section above.
